How FallSafe Began

Origin

FallSafe started as part of a Cloud Native Architecture and Development module. The module focuses on building real-world cloud-native systems, covering:

  • Backend services with Golang
  • Cloud platforms (AWS & Azure)
  • Containerisation with Docker
  • Scalable deployment using Kubernetes

What began as coursework evolved into a solution addressing a critical community need.

Dell Innovate Dash 2025

Developed during Dell Innovate Dash 2025, organised by Dell Technologies in collaboration with Lions Befrienders.

The team built a Cloud-Native, AI- and sensor-driven solution that enables fall-risk assessments using only a mobile phone.
